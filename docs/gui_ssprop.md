# Server set properties dialog

The Server set properties dialog is used to create a new server set, or to edit the properties of an existing server set.

You can create a new server set by clicking the "New set server..." button in the [main window - Server Set tab](gui_main_sstab.md).

You can edit the properties of an existing server from the  [main window - Server Set tab](gui_main_sstab.md) by selecting a server set in the left list and then clicking the "Properties..."  button in the toolbar.

The Server Set properties dialog consists of 8 sections:

- [Name](gui_ssprop_name.md)

- [Servers](gui_ssprop_servers.md)

- [Polling method](gui_ssprop_pollingmethod.md)

- [Polling frequency](gui_ssprop_pollingfreq.md)

- [Notification](gui_ssprop_notification.md)

- [DNS data](gui_ssprop_dnsdata.md)

- [Update DNS](gui_ssprop_updatedns.md)

- [Notes](gui_ssprop_notes.md)
