# Server properties dialog

The Server properties dialog is used to add a new server, or to edit the properties of an existing server within a server set.

You can create a new server by clicking the "Add..." button in the [Server set properties dialog / Servers section](gui_ssprop_servers.md).

You can edit the properties of an existing server from the [Server set properties dialog / Servers section](gui_ssprop_servers.md).selecting a server in the list and then clicking the "Edit..."  button.

The Server properties dialog consists of 2 sections:

- [General](gui_servprop_general.md)

- [Notes](gui_servprop_notes.md)
