# Main window

The main window in Simple Failover consists of a File menu and 3 tabs (Server Sets, Log, Problems), a Help button (blue circle with ?), and a tool bar:

![](https://jhsoftware.dk/is/5c8db9d3998f.png)

 ## File menu

- **Options...**\
Opens the [Options dialog](gui_opt.md) to configure general settings of the program (settings not related to any specific server set).

- **Help content and index**\
Opens this help file.

- **Online support...**\
Opens the web-page for the Simple Failover support web-site.

- **Support file...**\
Generates a support file with details about the program configuration for use by support staff.

- **Check for updates...**\
Checks the Simple Failover web-site to see if you are running the latest version / build.

- **Activate license...**\
Enter your license key to activate your software license.

- **Shutdown Simple Failover service...**\
Shuts down the service as well as the UI.

- **Exit Simple Failover UI**\
Shuts down the Simple Failover users interface, but leaves the Simple Failover service running.

## Tabs

For details see [Server Sets tab](gui_main_sstab.md), [Log tab](gui_main_logtab.md), [Problems tab](gui_main_probtab.md)

## Help button (blue circle with ?)

Opens this help file.

## Tool bar

The first 3 buttons on the tool bar are the same on all 3 tabs:

- **Options...**\
Opens the [Options dialog](gui_opt.md) to configure general settings of the program (settings not related to any specific server set).

- **Pause all polling**\
Pauses/resumes server polling for all server sets.

  **Poll all server sets**\
Polls all server sets immediately - resetting polling timers.
