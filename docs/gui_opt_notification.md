# Options dialog - Notification

In this section of the Options dialog, you can configure the default notification methods.

![](https://jhsoftware.dk/is/41158338c8a7.png)

 - **Default notification methods**\
List of default notification methods.\
These notification methods are used for server sets where the "Use default notification methods" option is enabled in the [Server set properties dialog / Notification section](gui_ssprop_notification.md).\
Use the *Add* button to add a new notification method - see [Notification properties dialog](gui_notify_method.md).\
Use the *Edit* button to update the properties of a notification method  - see [Notification properties dialog](gui_notify_method.md).\
Use the *Remove* button to remove a server.

 - **E-mail notification defaults... button**\
Click to open the [E-mail notification defaults dialog](gui_emaildef.md).
