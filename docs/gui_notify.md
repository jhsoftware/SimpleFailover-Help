# Notification properties dialog

The Notification properties dialog is used to create a new notification, or to edit the properties of an existing notification.

You can create a new notification by clicking the "Add..." button in the [Server set properties dialog / Notification section](gui_ssprop_notification.md) or in the [Options dialog / Notification section](gui_opt_notification.md).

You can edit the properties of an existing notification from the [Server set properties dialog / Notificaiton section](gui_ssprop_notification.md) or the [Options dialog / Notification section](gui_opt_notification.md) selecting a server in the list and then clicking the "Edit..."  button.

The Notification properties dialog consists of 3 sections:

- [Notify method](gui_notify_method.md)

- [Events](gui_notify_events.md)

- [Notes](gui_notify_notes.md)
