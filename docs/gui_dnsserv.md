# DNS server properties dialog

The DNS server properties dialog is used to add a new DNS server, or to edit the properties of an existing DNS server.

You can create a new DNS server by clicking the "Add..." button in the [Server set properties dialog / Update DNS section](gui_ssprop_updatedns.md) or in the [Options dialog / DNS updating section](gui_opt_dnsupdating.md).

You can edit the properties of an existing DNS server from the [Server set properties dialog / Update DNS section](gui_ssprop_updatedns.md) or the [Options dialog / DNS updating section](gui_opt_dnsupdating.md) selecting a DNS server in the list and then clicking the "Edit..."  button.

The DNS server properties dialog consists of 4 sections:

 - [Server name / IP](gui_dnsserv_nameip.md)

 - [DNS lookup](gui_dnsserv_dnslookup.md)

 - [Update method](gui_dnsserv_update.md)

 - [Notes](gui_dnsserv_notes.md)
