# Server set properties dialog - Servers

In this section of the Server set properties dialog, you can specify the individual servers that make up the server set:

![](https://jhsoftware.dk/is/547348380a54.png)

 - **Servers**\
List of individual servers that make up the server set.\
Use the *Add* button to add a new server - see [Server properties dialog](gui_servprop_general.md).\
Use the *Edit* button to update the properties of a server - see [Server properties dialog](gui_servprop_general.md).\
Use the *Remove* button to remove a server.\
Use the *Move Up* and *Move Down* buttons to change the server polling order.
