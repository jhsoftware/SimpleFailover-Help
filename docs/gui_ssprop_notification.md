# Server set properties dialog - Notification

In this section of the Server set properties dialog, you can specify how to be notified of problems / events related to this server set:

![](https://jhsoftware.dk/is/72b71e1a1eb1.png)

 - **Use default notification methods (configured in Options dialog)**\
Enable this to send notifications for this server set using the default notification methods configured in the [Options dialog / Notification section](gui_opt_notification.md).

 - **Notify of problems/events using the following methods (for this server set only):**\
List of notification methods for this server set.\
Use the *Add* button to add a new notification method - see [Notification properties dialog](gui_notify.md).\
Use the *Edit* button to update the properties of the selected notification method  - see [Notification properties dialog](gui_notify.md).\
Use the *Remove* button to remove the selected notification method.
