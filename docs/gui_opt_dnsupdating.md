# Options dialog - DNS updating

In this section of the Options dialog, you can configure general settings related to DNS updates.

![](https://jhsoftware.dk/is/cfb7104a4af0.png)

- **Max. concurrent DNS updates**\
Defines the maximum number of DNS updates to perform at the same time. 

- **Default DNS servers to update**\
List of default DNS servers to update.\
These DNS server are used for server sets where the "Update the default DNS servers" option is enabled in the [Server set properties dialog / Update DNS section](gui_ssprop_updatedns.md)\
Use the Add button to add a new DNS server - see [DNS server properties dialog](gui_dnsserv.md).\
Use the Edit button to update the selected DNS server - see [DNS server properties dialog](gui_dnsserv.md).\
Use the Remove button to remove a DNS server.
