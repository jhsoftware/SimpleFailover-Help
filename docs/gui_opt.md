# Options dialog

The Options dialog is used to configure general settings - not related to any specific server set.

You reach the Options dialog from the [main window](gui_main.md) by clicking the "Options..." button.

The Options dialog consists of 7 sections:

- [User interface](gui_opt_ui.md)

- [Server polling](gui_opt_serverpolling.md)

- [Notification](gui_opt_notification.md)

- [DNS updating](gui_opt_notification.md)

- [DNS serving](gui_opt_dnsserving.md)

- [Logging](gui_opt_logging.md)

- [HTTP API](gui_opt_httpapi.md)
