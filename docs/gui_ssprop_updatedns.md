# Server set properties dialog - Update DNS

In this section of the Server set properties dialog, you can specify which DNS servers to update for this server set (whenever a server fails or comes back online).

![](https://jhsoftware.dk/is/797382d414f4.png)

- **Update the default DNS servers (configuration in Options dialog)**\
Enable this to update the default DNS servers configured in the [Options dialog / DNS updating](gui_opt_dnsupdating.md) section for this server set.

- **Update the following DNS servers (for this server set only)**\
List of DNS servers to update for this server set.\
Use the *Add* button to add a new DNS server - see [DNS server properties dialog](gui_dnsserv.md).\
Use the *Edit* button to update the selected DNS server - see [DNS server properties dialog](gui_dnsserv.md).\
Use the *Remove* button to remove the selected DNS server.
